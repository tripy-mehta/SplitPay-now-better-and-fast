# User feedback summary

Collected via the in-app feedback prompt (shown after a user completes their
first on-chain settlement) and tracked as `feedback_submitted` events in
PostHog.

## Method

- 10+ testers were given the production URL and asked to: connect Freighter,
  fund via Friendbot, create or join a group, log at least one real expense,
  and complete a settlement.
- After settling, each saw a 3-question in-app prompt: clarity rating,
  "would you use this with real money?", and an optional open comment.
